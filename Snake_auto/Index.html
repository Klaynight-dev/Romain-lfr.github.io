Snake Automatique

Ce Projet est la suite du premier Projet il a pour but de l'automatiser

Dans le cadre de mon premier projet, j'ai développé un jeu de Snake interactif où le mouvement du serpent était guidé par l’utilisateur. Dans ce projet, j'ai rendu le serpent autonome, c'est-à-dire que j'ai programmé ses mouvements afin qu’il mange les pommes le plus rapidement possible.

Pour juger de l’efficacité de mes programmes, ceux-ci afficheront le nombre de déplacements unitaires, ainsi que le temps CPU.
